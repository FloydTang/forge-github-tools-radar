# forge-github-tools-radar 🛠️⚡

> Forge 专属 - GitHub 开源工具巡视与深度分析

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.10+](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)

**forge-github-tools-radar** 是半斤九两科技 Forge 子代理的专属 Skill，实现 GitHub 开源工具的自动巡视、筛选与深度分析。

---

## ✨ 特性

| 特性 | 说明 |
|------|------|
| **定时巡视** | 每天自动搜索 GitHub 相关工具 |
| **关键词库** | TikTok/社媒/出海相关工具关键词 |
| **5维度评分** | 落地相关性、易用性、免费/开源、社区活跃、可扩展 |
| **深度分析** | 👍 触发后进行详细分析 |
| **自动化工作流** | 巡视 → 筛选 → 发布 → 深度分析 |

---

## 🎯 价值与意义

### 解决什么问题？

**传统工具调研的痛点**：
- 手动搜索效率低，难以发现优质工具
- 缺乏系统化评估标准
- 调研结果分散，难以复用

**forge-github-tools-radar 的答案**：
- 自动化巡视 + 标准化评分，效率提升 10x
- 5 维度评分，决策更客观
- 深度分析能力，定制开发更精准

### 适用场景

- 🔍 开源工具调研
- 🛠️ TikTok/社媒工具发现
- 📊 自动化工具评估

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/FloydTang/forge-github-tools-radar.git
cd forge-github-tools-radar
```

### 2. 配置

```bash
# 配置 GitHub Token（可选，用于更高 API 限流）
export GITHUB_TOKEN="your_github_token"
```

### 3. 运行

```bash
# 查看 SKILL.md 了解完整工作流程
cat SKILL.md
```

## 📊 评分标准

| 维度 | 权重 | 说明 |
|------|------|------|
| 落地相关性 | 30% | 与落地营8周内容的直接关联度 |
| 易用性 | 20% | 安装/配置/使用门槛 |
| 免费/开源 | 15% | 是否免费、是否有开源版 |
| 社区活跃 | 20% | Stars、更新频率、文档质量 |
| 可扩展 | 15% | 能否二次开发、定制改造 |

**总分 ≥ 3.5 分** → 值得分享
