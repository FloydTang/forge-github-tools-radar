# Forge GitHub 工具雷达

## 任务概述

每天在 GitHub 搜索与 TikTok 落地营相关的开源工具，整理发布到频道；检测到 👍 后深度分析。

## 搜索关键词库

```
tiktok, video-tools, short-video, tiktok-api, tiktok-scraper
social-media-tools, video-editor, content-scheduler
advertising-tools, ad-analytics, facebook-ads-api
digital-human, ai-avatar, talking-head, wav2lip
video-translation, subtitle-generator, caption-generator
instagram-tools, youtube-tools, social-media-analytics
```

## 筛选标准

### 基础筛选：与落地营 8 周内容相关

| 周 | 相关工具 |
|----|---------|
| 第1周 | IP 检测、伪装度检测、账号检测、代理池 |
| 第2周 | 视频剪辑、批量处理、素材采集、去重 |
| 第3周 | 账号分析、流量分析、推荐算法分析 |
| 第4周 | 封面设计、Banner 制作、Canva API |
| 第5周 | 广告投放工具、竞品分析、投放管理 |
| 第6周 | 数字人、AI Avatar、语音合成、口型同步 |
| 第7周 | 表单构建、CRM、邮件自动化、WhatsApp API |
| 通用 | 社媒排程、数据看板、自动化脚本 |

### 进阶筛选：社媒数据分析

**额外关注**：
- 竞品社媒数据分析工具
- 自有账号数据监控工具
- 知乎/LinkedIn 数据抓取工具
- 多平台数据汇总看板

## 工具评分标准（5维度）

**总分 ≥ 3.5 分** → 值得分享

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **落地相关性** | 30% | 与落地营8周内容的直接关联度 |
| **易用性** | 20% | 安装/配置/使用门槛有多低 |
| **免费/开源** | 15% | 是否免费、是否有开源版 |
| **社区活跃** | 20% | Stars、更新频率、文档质量 |
| **可扩展** | 15% | 能否二次开发、定制改造 |

### 评分格式

```
- **评分**: X.X/5 (落地X + 易用X + 免费X + 活跃X + 扩展X)
```

## 工具分类标签

| 标签 | 含义 |
|------|------|
| 🛠️ 运营工具 | 直接可用于日常运营 |
| 📊 数据分析 | 竞品/自有数据监控 |
| 🤖 自动化 | 批量处理、自动化工作流 |
| 💡 创意内容 | 视频/图片创作辅助 |
| 🔧 技术向 | 需要技术背景才能使用 |
| ⭐ 强烈推荐 | 4.5分以上，必看 |

## 输出格式

```markdown
## 🔧 今日工具发现 - YYYY-MM-DD

### [工具名称]
- **评分**: X.X/5 (落地X + 易用X + 免费X + 活跃X + 扩展X)
- **用途**：一句话说明
- **GitHub**：repo 链接
- **Stars**：xxx
- **适合**：第x周 / 通用
- **标签**：🛠️运营工具 📊数据分析 等
- **一句话评价**：xxx
```

**总分 ≥ 3.5 分的工具才发布**，低于此标准的可以记录但不发布。

## 发布位置

- **日常发布**：`#🛠️ TikTok落地工具` (1482079207528861756)
- **深度分析**：`#锻造舰·代码库` (1480661441672515674)

## 工作流程

### 日常巡视（每日 04:00 UTC）

1. 用关键词在 GitHub 搜索（可用 Trending 或 Search API）
2. 筛选近期更新、Stars 较高、与落地营相关的项目
3. 每个工具整理成上述格式
4. 发布到 #🛠️ TikTok落地工具

### 深度分析（检测到 👍 触发）

1. 检测 #🛠️ TikTok落地工具 的 👍
2. 对被 👍 的工具进行深度调研：
   - 核心技术栈
   - 安装部署方式
   - 能否定制改造
   - 与落地营的结合点
3. 在 #锻造舰·代码库 发布分析报告：
   ```markdown
   ## 🔍 工具深度分析 - [工具名]

   ### 基本信息
   - GitHub: xxx
   - Stars: xxx
   - 评分: X.X/5
   
   ### 功能详情
   [详细功能描述]
   
   ### 定制可行性
   - 难度：⭐⭐⭐⭐⭐
   - 改造建议：xxx
   
   ### 与落地营结合
   [如何应用到课程中]
   ```
4. 等待九两讨论下一步

## 注意事项

- 每条工具单独一条消息，方便九两单独 👍
- 优先选择近期有更新的项目（6个月内）
- 优先选择文档清晰的项目
- 简要评价即可，详细分析留给 👍 触发后
